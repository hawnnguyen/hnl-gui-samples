Sample hnl-enhanced-jsf-webapp with HTML5, CSS3, jQuery, internationalization (i18n) simple login,  with cool effects!!
============================
1.  Install Maven and Java
2.  On the Root Directory run  "mvn clean install"
3.  Then run "mvn tomcat:run-war" to startup the server
4.  You can access the webapp with:  

http://localhost:8080/hnl-enhanced-jsf-webapp
